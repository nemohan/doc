# 流量捕获

[TOC]

流量捕获的几种方案:

* gopacket 基于libpcap
* sflow 独立二进制。（底层仍是基于libpcap和ebpf)
* ebpf 基于内核，内核版本有要求（貌似是高于3.15）



## gopacket

gopacket 也有三种方案捕获流量:

* af_packet     **AF_PACKET** is a new feature in Linux 2.2
* pf_ring
* pcap (性能差)

若默认情况下使用动态库libpcap.so.0.8

在ubuntu使用如下命令安装：

~~~
apt -y install libpcap-dev
~~~

在centos安装:

~~~
yum -y install libpcap-devel
~~~

以静态库的方式集成libpcap:

~~~
CGO_LDFLAGS+="-Wl,-static -L/usr/lib/x86_64-linux-gnu/libpcap.a -lpcap -Wl,-Bdynamic" go build packet.go

CGO_LDFLAGS+="-Wl,-static -L/usr/lib/x86_64-linux-gnu/libpcap.a -lpcap -Wl,-Bdynamic" go build packet.go
~~~





### 基于pcap

~~~go
package main


import(
	"time"
	"fmt"
	"flag"

	"github.com/google/gopacket"
	"github.com/google/gopacket/layers"
	"github.com/google/gopacket/pcap"
)

var (
	device = flag.String("d", "", "link device")
	ethLayer layers.Ethernet
	ipLayer layers.IPv4
	tcpLayer layers.TCP
)
func main(){
	flag.Parse()
	if *device == ""{
		fmt.Printf("no device\n")
		return
	}
	handle, err := pcap.OpenLive(*device, 1024, false, time.Second * 30)
	if err != nil{
		panic(err)
	}
	defer handle.Close()
	packetSource := gopacket.NewPacketSource(handle, handle.LinkType())
	for packet := range packetSource.Packets(){
		parser := gopacket.NewDecodingLayerParser(
			layers.LayerTypeEthernet,
			&ethLayer,
			&ipLayer,
			&tcpLayer,
		)

		foundLayerTypes := []gopacket.LayerType{}
		err := parser.DecodeLayers(packet.Data(), &foundLayerTypes)
		if err != nil{
			fmt.Printf("decoding layers failed. %v\n", err)
		}
		for _, layerType := range foundLayerTypes{
			if layerType == layers.LayerTypeIPv4{
				fmt.Printf("ipv4: src:%v dst:%v\n", ipLayer.SrcIP, ipLayer.DstIP)
			}
		}
	}

}

~~~



### 基于af_packet

~~~go
package main

import(
	"fmt"
	_ "github.com/google/gopacket"
	"github.com/google/gopacket/afpacket"

)

func main(){
	handle, err := afpacket.NewTPacket(afpacket.OptInterface("eth0"))
	if err != nil{
		panic(err)
	}

	i := 0
	defer handle.Close()
	for{
		_, capInfo, err := handle.ReadPacketData()
		if err != nil{
			fmt.Printf("%v\n", err)
		}
		if i == 100000{
			fmt.Printf("%d\n", capInfo.Length)
			i = 0
		}
		i++

	}

}
~~~
