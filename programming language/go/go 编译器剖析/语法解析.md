# 语法解析部分



重点关注parser处理词法和语法错误的机制



##### parser.got

~~~go
func (p *parser) got(tok token) bool {
	if p.tok == tok {
		p.next()
		return true
	}
	return false
}
~~~



##### parser.want

~~~go
func (p *parser) want(tok token) {
	if !p.got(tok) {
		p.syntax_error("expecting " + tok.String())
		p.advance()
	}
}
~~~
