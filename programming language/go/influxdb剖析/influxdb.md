# influxdb剖析

新入职的公司主要是做数据监控及分析的，期间用到influxdb存储时序数据。借此机会学习一下influxdb的内部实现。主要想学习下面两点：

* 数据存储引擎的实现
* 查询语句的解析

目前influxdb的版本是2.0。为了简单起见，选择1.8.3版本作为起点。
