## tsm文件格式

~~~
magic(4字节) | version(1字节)
++++++++++++++++++++++++++++++++++++++++++++++++++++++++++++++++++++++++++++++++++++
block_1 | block_2
++++++++++++++++++++++++++++++++++++++++++++++++++++++++++++++++++++++++++++++++++++
index | index_2
++++++++++++++++++++++++++++++++++++++++++++++++++++++++++++++++++++++++++++++++++++
index offset(8字节)即index 在当前文件的起始存储位置
++++++++++++++++++++++++++++++++++++++++++++++++++++++++++++++++++++++++++++++++++++

block格式:
crc(4字节) | 块数据
+++++++++++++++++++++++++++++++++++++++++++++++++++++++++++++++++++

index格式:
key 的长度(2字节) | key | block type(1字节) | block 数目| entry_1| entry_2
+++++++++++++++++++++++++++++++++++++++++++++++++++++++++++++

index entry格式:
min_time(8字节) | max_time(8字节) | block offset (8字节) | block size(4字节)
+++++++++++++++++++++++++++++++++++++++++++++++++++++++++++++

magic:
0x16D116D1

version:
1
~~~



## tsm存储引擎架构

tsm存储引擎包括四个主要模块:

* cache: 数据先写入缓存，然后再由comapct模块将其刷到磁盘
* encoding: 数据编码
* wal: 数据写入wal同时写入缓存，防止数据丢失
* compact: 将缓存数据写入磁盘，同时定期将多个tsm文件合并
