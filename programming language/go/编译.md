## 静态连接

使用动态连接库时，将二进制分发到不同的平台经常会遇到xx.so未找到的问题。因此需要使用静态连接连接动态库对应的静态库。在golang中可以使用CGO_LDFLAGS参数指定连接对应的静态库。如下所示

~~~rust
CGO_LDFLAGS+="-Wl,-static -L/usr/lib/x86_64-linux-gnu/libpcap.a -lpcap -Wl,-Bdynamic" go build xxx
~~~



## 使用ldflags设置go程序的版本

~~~shell
GOOS=linux go build -ldflags "-X 'main.branch=$(git rev-parse --abbrev-ref HEAD)' -X 'main.commit=$(git rev-parse --short HEAD)'"
~~~



## 跨平台编译

在go build命令之前加上“GOOS=目标平台”参数可以进行跨平台编译

~~~shell
GOOS=linux go build xxx
~~~
