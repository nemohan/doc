# 标记压缩算法

### 标记压缩算法

算法描述:

重新移动活动对象，使得每个活动对象之间不存在空闲空间

~~~
压缩前:
|-----------|-----------|-----------|-----------|--------------
|used		|			|used		|			|used
|	     	|			|			|			|
|-----------|-----------|-----------|-----------|---------------

压缩后:
|-----------|-----------|-----------|-----------|--------------
|used		|used		|used		|			|
|	     	|			|			|			|
|-----------|-----------|-----------|-----------|---------------
~~~
